# GlideMultiDiskCacheDemo

This sample demonstrates how to support setting multiple disk cache directories for [Glide][1].

此项目是一个Android Glide扩展演示Demo，支持给Glide设置多个磁盘缓存目录，并在加载时指定缓存位置。
